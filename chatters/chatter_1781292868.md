## 什么是agent？和GPT有什么区别？

如果说GPT是脑子（因为你让他帮你想东西），那agent就是身体。你有GPT只能想东西说东西，但是你有了agent可以把脑子里的信息转化成实物，可以用脑子去造火箭

比如 GPT不能帮你做PPT，不能帮你整理表格，他只能看完以后告诉你他的感觉

但是接入agent以后他可以给你改PPT，给你做表格，甚至帮你把PPT发给你的导师

  
‍

假设你说：

👉 “帮我修复这个Python bug”

* ChatGPT：  
  “第12行可能有问题，建议这样修改……”
* agent：
* “我已经修复并通过测试，这是修改结果，要不要提交？”

  
‍  
‍

## codex是什么？

OpenAI开发的一个agent，在我看来这是世界上最好用的agent，他有一双健壮的臂膀（工作很踏实，做出来的东西让人满意）

![](/image/pasted-1781291763939-ec5535.png)

## CC Switch是什么？

你想给你的agent接什么大脑？接入GPT5.5？还是deepseekV4pro？

不管你怎么接，都得把Key输入到agent里

但是怎么输入呢？你得改agent的配置文件

我看不懂配置文件怎么办？

——用CC Switch

他可以帮你改配置文件，你直接往里面输入Key就可以自动改Codex或者Claude code 或者Openclaw等等的配置的Key

  
‍

## 如何使用？

首先下载好CC Switch后进入配置并关闭所有路由

![](/image/pasted-1781292473408-e078d4.png)

然后这里可以打开一下codex应用增强

![](/image/pasted-1781292532299-e6087b.png)

其他的不用管就这样吧（我很少用Claude，等用的时候再跟大家讲）

接下来加入你的Key

选OpenAI（也就是第一个箭头指向） 然后加号

![](/image/pasted-1781292626380-8e318b.png)

然后 自定义配置 名称和备注任意 官方链接不写 下面填入你的API Key和你的API URL

![](/image/pasted-1781292677765-c95aa9.png)

如果你要接入非GPT的模型的话才打开那个本地路由映射

然后保存

出来以后可以测试一下模型能不能用

![](/image/pasted-1781292783335-836b21.png)

这些搞完之后使用这个

![](/image/pasted-1781292813992-adba97.png)

然后打开Codex即可，Codex已经接入了你的API Key
