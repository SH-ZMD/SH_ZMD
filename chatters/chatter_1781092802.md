## GPT模型是什么

OpenAI公司的GPT系列模型和Anthropic公司的Claude opus模型（我最熟悉的两个，除此之外还有马斯克的grok等等）

## API接口

API是什么呢？比如你想在QQ里接入一个机器人，这个机器人内置gpt-5.5模型，该怎么办？难道让机器人随时在ChatGPT网页对话吗？不是的。

OpenAI开放了一个接口，就是API。

API当作一个通道，你给这个QQ机器人设置了API通道/API地址（URL）和API key(API密钥)，然后机器人收到请求以后会通过这个API通道直接把消息发给OpenAI官方让OpenAI官方给你回消息，不通过网页或者什么别的。

![](/image/pasted-1781091349247-9c8581.png)

## API URL是什么?

URL是地址的意思，简单来说你给你的QQ机器人内置gpt模型，总得让机器人知道从哪里获取OpenAI的消息，所以需要给他一个链接来让他明白去哪个链接找OpenAI要大模型数据

## **API Key 是什么？**

API Key 就当作密码。

你给QQ机器人接入了地址不行，OpenAI不能让你白用人家大模型。

你要让某个软件调用 AI，它总得知道是谁在用、用谁的额度、有没有权限。  
这个时候就需要 API Key。

你把 API Key 填进软件里，软件就可以拿着这把密码登录OpenAI账号去请求 AI 服务。

  
‍

现在好多软件可以接入API Key来调用大模型，甚至我的世界里的车万女仆都可以接入API Key来让你的女仆可以和你说话，这就是API Key的方便性

![](/image/pasted-1781092608842-43ad1e.png)

## Token是什么？

你使用API Key肯定也要交钱，这些钱就叫Token，Token其实你可以当作字节，gpt分析了100个字节，就花费300token这个样子，然后你就需要买token（因为这些都会花OpenAI公司的电费）

![](/image/pasted-1781091640430-4c44e1.png)

## 怎么便宜得到token？
